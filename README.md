# Quick Start

## Requirements

- Node.js 18+
- npm

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal ( usually http://localhost:5173/layer-list/ ).

## Deployment

GitHub Pages: https://sagewall.github.io/layer-list/

## Other Useful Commands

```bash
npm run build
npm run preview
```
